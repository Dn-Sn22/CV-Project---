# CV Project: Object Detection / Computer Vision

## Описание проекта
Данный проект посвящён обучению нейронной сети для задачи компьютерного зрения.
Работа выполнялась в среде Google Colab.

Цель проекта — изучить полный пайплайн работы с CV-задачей:
от подготовки данных до обучения модели и оценки качества.
(Обнаружение и подсчет количества людей на видео)

---

## Используемые технологии
- Python
- Google Colab
- NumPy
- Pandas
- OpenCV
- Matplotlib
- Albumentations
- YOLO

## Подход и архитектура
В рамках проекта была использована модель **YOLOv8**.

Основные этапы:

***Первый пайплайн***
1. Предобработка изображений
2. Разметка и подготовка данных
3. Обучение модели
4. Валидация и тестирование
5. Анализ результатов

***Второй пайплайн***
1. Предобработка изображений
2. Разметка и подготовка данных
3. Обучение модели
4. Валидация и тестирование
5. Анализ результатов

---

## Метрики и результаты
Для оценки качества модели использовались следующие метрики:
- Accuracy
- Precision
- Recall
- mAP

Достигнутые результаты:
- Accuracy: 0.85
- Recall: 0.65
- mAP: 0.76
- mAP@50-95: 0.51

(Значения могут обновится по мере улучшения модели)
